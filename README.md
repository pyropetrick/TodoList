# Todo List

## Install

- install docker: [docs](https://docs.docker.com/engine/install/)

- clone repository: ```git clone git@github.com:pyropetrick/TodoList.git```

## Build and start

### Start

```bash
docker compose up
```

### Stop

```bash
docker compose down
```

### URL

Open frontend: ```http:localhost:4000```

Backend docs: ```http:localhost:3000/api/docs```
